# Major file types

## JSON

### Map object
```
{
    "name": "Captain Underpants",
    "f_name": "Captain",
    "l_name": "Underpants",
    "age": 10,
    "occupation": "World Domination"
}
```

### Lists

#### Integer List
```
[1, 2, 3, 4, 5, 6]
```

#### String list
```
['a', 'b', 'c', 'd']
```

#### List of Maps (Pretty)
```
[ 
    {
        "name": "Captain Underpants",
        "age": 10,
        "hometown": "America"
    },
    {
        "name": "Kal-El",
        "age": 30,
        "hometown": "Krypton"
    }
]
```

#### List of Maps (Condensed)
```
[
    {"name": "Captain Underpants", "age": 10, "hometown":"America"},
    {"name": "Kal-El", "age": 30, "hometown": "Krypton"}
]
```
